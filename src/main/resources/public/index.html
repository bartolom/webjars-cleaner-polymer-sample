Browser Links

This web page demonstrates that we can resolve typical Polymer dependencies with and without the prefix like github-com-PolymerElements-

When you follow the links you will in most cases see a blank page, which indicates a HTTP 200 (use the browsers view source feature). If you click this in a browser that supports natively HTML import (Chrome) you can see that the whole dependcy tree is pulled
